# 조건문



## if [condition] ~\~\~ do \~\~\~ end

```ruby
if true
	puts 'true'
end
puts 'anyway'
```



## if [condition] ~\~\~ else \~\~\~ end

```ruby
x = 1
if x > 2
	puts 'if'
else
	puts 'else'
end
puts 'end'
```



## if [condition] \~\~\~ elseif [condition2] \~\~\~ else \~\~\~ end

```
x = 1
if x > 2
	puts 'if'
elseif x > 1
	puts 'elseif'
else
	puts 'else'
end
puts 'end'
```
